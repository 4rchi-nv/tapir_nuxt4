# Tapir — тестовое задание Frontend

Nuxt 4 + TypeScript приложение: главная страница и каталог товаров с пагинацией «Показать ещё».

## Технологии

- Nuxt 4
- TypeScript
- SCSS, БЭМ
- SSR

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Сборка

```bash
npm run build
npm run preview
```

## Функциональность

- **Шапка**: ссылки-заглушки (`href="#"`), бургер без действия.
- **Главная**: кнопка «В каталог» ведёт на `/catalog`.
- **Каталог**:
  - первая страница загружается при открытии (SSR);
  - карточка товара — ссылка-заглушка;
  - кнопка «Добавить в избранное» только визуальная;
  - цена x2 = `ceil(цена / 2)`;
  - «Показать ещё» — подгрузка следующей страницы, при ошибке блок «Повторить».
- **Подвал**: ссылки-заглушки.

API: `https://test-task-api.tapir.ws/products?page=1&limit=10`
